## 6. 謝里法捐贈案例：完整流程（業務流程 × 系統功能）

### 6-1 收到捐贈意向與建立徵集審議案件

- **入口**：徵集審議作業 → 新增案件／案件列表
- **重點**：建立案件、上傳清單附件、設定年度/承辦/狀態

### 6-2 評估作業：初評/複評與分級

- **入口**：徵集審議作業 → 案件內作業
- **重點**：評估表、分級、意見彙整、統計

### 6-3 審議作業：委員、會議、決議

- **入口**：徵集審議作業 → 案件內作業
- **重點**：審議流程、委員設定、會議紀錄、審查表單與統計表、決議與條件

### 6-4 取得作業：來源登記與條款摘要

- **入口**：徵集審議作業 → 案件內作業（取得作業）
- **重點**：取得方式（捐贈/移撥/價購…）、原所有者/單位、條款摘要

### 6-5 建立全宗架構（謝里法全宗 → 系列/卷/件層）

- **入口**：全宗架構管理 → 架構維護
- **重點**：建立系列 A（藝術家資料 627）、系列 B（錄音錄影 26）、系列 C（藝術類文獻）與件層清單

### 6-6 入庫點收、館藏號編碼、保存地/位置

- **入口**：入庫作業管理
- **重點**：入庫批次/點收、差異比對、異常處理、館藏號規則、保存地/位置

### 6-7 保存狀況分級與保存修護作業（新增）

- **入口**：入庫作業管理 → 保存修護
- **重點（狀況等級）**：
    - 依單位規則建立狀況等級（例：1～4 級；1最佳、4最差）
    - 於入庫/整理時為實體物件登錄初始狀況等級與描述，保留等級歷程
- **重點（啟動修護）**：
    - 建立修護作業單（可由入庫異常觸發或主動建立）
    - 從全宗架構挑選或搜尋物件名稱/館藏號加入（可快速篩選 3/4 級）
    - 指派修護人/單位、起迄日期、優先序（選配：預估費用、送修/回收資訊）
    - 修護回來後上傳整體修護記錄文件；每一筆物件可選擇加註修護說明並調整等級

### 6-8 數位化案件：上傳、對應、QC

- **入口**：數位化作業 → 案件列表 → 案件內流程
- **重點**：原始/數位化檔分層、縮圖預覽、圖文/影音對應、QC

### 6-9 待編目登記（進度追蹤與可上架狀態）

- **入口**：詮釋資料維護 → 待編目登記
- **重點**：待編目清單、進度追蹤、可上架狀態

### 6-10 編目作業（含 OCR/文字擷取）

- **入口**：詮釋資料維護 → 編目作業
- **重點**：metadata 建置、日期/來源/權利來源註記；OCR 觸發→回存→修正

### 6-11 資料類型設定（欄位範本 Template 套用）

- **入口**：詮釋資料維護 → 資料類型設定
- **重點**：預載欄位範本、配置必填/多值/型態/顯示

### 6-12 權利設定與存取限制落地

- **入口**：系統設定 → 權利設定
- **重點**：權利盤點、CC 註記、存取限制規則（影響前台顯示/下載/提借）

### 6-13 AI 與索引（RAG）

- **入口**：系統設定 → AI 與索引設定
- **重點**：向量索引建立/更新、索引任務監控

### 6-14 網站上稿與前台應用

- **入口**：網站管理
- **重點**：建立「謝里法特藏」主題頁、欄位露出、外部上稿審核與異動稽核

### 6-15 提借(閱)申請：實體/數位/影音

- **入口**：提借(閱)作業 → 申請案件列表
- **重點**：案件管理、簽核流程（可配置）、案件狀態歷程、通知（由參數設定驅動）
