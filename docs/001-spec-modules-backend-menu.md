## 

> 本文件定位：系統功能規劃（模組與能力）＋後台選單 IA（表格）。
> 

---

## 1. 系統核心與願景（Core Vision）

- **核心資料主體**：典藏單元（Item）與其階層（全宗/系列/卷）＋數位資產（檔案）＋詮釋資料（metadata）＋實體保存狀態/修護歷程
- **解決的問題**：
    - 從徵集/審議 → 入庫 → 保存修護 → 數位化 → 編目/審核 → 發佈 → 前台查詢 → 提借(閱)申請，形成可追溯、可控的一條龍作業
    - 支援 AI 協助編目與 OCR；建立向量索引（RAG）供前台語意搜尋
    - 以權利與存取限制控管前台露出、下載、申請與引用

## 3. 系統功能規劃（按模組）

### 3.1 整合查詢

- 單一查詢介面（關鍵字＋欄位條件）
- 可依後設資料欄位分項查詢、交叉檢索、進階條件組合

### 3.2 徵集審議作業

- 新增案件、案件列表（依年度/狀態）
- 案件內作業：評估作業、審議作業、選擇審議委員、線上會議、取得作業、審查表單與統計
- 匯入匯出（Excel/Word 對應＋檢核）

### 3.3 全宗架構管理

- 架構維護：同一樹狀架構編輯介面管理全宗/系列/卷/件層（含件層清單檢視/篩選/批次操作）
- 匯入匯出：結構/件層清單批次匯入匯出（Excel/Word 對應＋錯誤報表）

### 3.4 入庫作業管理（含保存修護）

### A. 入庫批次/點收

- 到館點收、差異比對、狀態標示、異常處理（缺件/破損/需修復）

### B. 館藏號/編碼規則

- 系統編碼 vs 自定義編碼、查重、發號

### C. 庫房/保存地管理

- 保存地/位置階層、調閱流向

### D. 保存狀況分級（Condition Rating）

- **依單位/典藏類別可設定不同等級規則**（例：1～4 級；1最佳、4最差）
- 實體物件於入庫/整理時可登錄：
    - 初始狀況等級、狀況描述（自由文字/選項）、檢視日期、檢視人
    - 狀況等級歷程（每次變更保留時間/人員/原因）

### E. 保存修護作業（修護作業單）

- **建立修護作業單**（Repair/Treatment Work Order）：
    - 來源：可由入庫點收異常觸發、或由保存修護人員主動建立
    - 選取待修護物件：
        - 從**全宗架構管理樹狀介面挑選**（多選）
        - 或使用**整合查詢**以物件名稱/館藏號搜尋並加入
        - 支援快速篩選：狀況等級＝3/4（或各單位定義的高風險等級）
    - 修護資訊：修護人（館內）/修護單位（外包）、起迄日期、優先序、預估費用（選配）、送修地點/方式（選配）
- **修護執行與回收**：
    - 作業單狀態：草稿 → 送核（選配）→ 進行中 → 待驗收 → 已完成 → 結案
    - 交付/回收紀錄（Chain of Custody）：送出日期、收回日期、簽收人
- **修護紀錄與回寫**：
    - 上傳整體修護記錄文件（PDF/圖片/附件）與前後對照照片（選配）
    - 針對每一筆物件：
        - 是否加註修護狀況說明
        - 是否調整狀況等級（例：4 → 2）
        - 修護摘要（可供前台顯示的版本需另行控管）
    - 修護歷程可追溯（物件層：修護單/附件/變更紀錄）

### 3.5 數位化作業

- 案件列表：依年度/狀態/承辦篩選
- 案件內流程：檔案上傳、原始/數位化檔分層、縮圖預覽、圖文/影音對應、QC
- 格式支援與檢核：JPEG/TIFF/MP4/PDF…（由系統設定白名單控管）

### 3.6 詮釋資料維護

- 待編目登記：審議完成但未編目資料集中管理、進度追蹤、可上架狀態
- 編目作業：單筆/批次編目、欄位編碼、人工/批次匯入匯出
- **OCR/文字擷取**：作為編目過程的功能（觸發、結果回存、修正）
- 資料類型設定：預載欄位範本（Template），設定各資料類型對應欄位（必填/多值/型態/顯示）

### 3.7 網站管理

- 網站架構、主題單元、版型、資料上稿
- 依權利/存取限制決定前台欄位露出與可用範圍
- 外部上稿審核與異動紀錄（稽核）

### 3.8 提借(閱)作業

- 申請案件列表：前台申請案件檢視；後台可新增申請單
- 申請項目：實體物件、數位檔、影音檔
- 審查流程：承辦→主管→主秘→副館長→館長（可配置）
- 案件狀態：出借中/歸還/結案（在案件內狀態與歷程處理）

### 3.9 資料統計及報表

- 詮釋資料欄位統計、狀態統計
- 前台統計：參訪人數、點閱率
- 可編輯報表與圖表、匯出
- 稽核紀錄：上稿/異動與關鍵操作 log

### 3.10 系統設定

- 帳號/角色/權限（含外部使用者）
- 閱覽/下載存取限制：依角色、館內外 IP、登入身分、授權程度
- 工作流程設定：審議、編目審核、提借審查
- 權利設定：權利盤點、CC 註記、存取限制、法規依據
- AI 與索引設定：AI 模型、向量索引、索引更新策略（RAG）
- 儲存/媒體設定：儲存桶、路徑、轉檔規則、格式白名單
- 參數設定：通知/EIP/email 介接參數、外部服務端點、共用字典參數
    - **保存狀況等級字典/規則（依單位/典藏類別）**
    - 修護作業單狀態字典、優先序字典（選配）
